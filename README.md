# Exec Dashboard Demo

A standalone, runnable version of the Exec Dashboard, built with Vite + React,
Tailwind CSS v4, Recharts, D3 (force-directed dependency graph), and
lucide-react icons. All data is embedded in `src/ExecDashboard.jsx`, so the app
runs entirely client-side with no backend.

## Getting started

```bash
npm install
npm run dev
```

Then open the printed local URL (defaults to http://localhost:5173/).

## Scripts

- `npm run dev` — start the dev server with hot reload
- `npm run build` — production build into `dist/`
- `npm run preview` — preview the production build locally
